# ALSUWEIDI ERP

Frontend-only CRM/HR proof-of-concept for ALSUWEIDI Engineering Consultants.

**→ Read [SPEC.md](SPEC.md) first.** It covers architecture, the full data model, feature map, UI conventions, and — importantly — the known gaps and risks to read before building a real backend against this.

**Live demo**: https://alsuweidi-erp-demo.pages.dev (login with any name, pick a role — no password)

## Quick start

```
cd frontend
npm install
npm run dev
```

Work from a local-disk clone, not a Google Drive-synced folder — see SPEC.md §1 for why.
